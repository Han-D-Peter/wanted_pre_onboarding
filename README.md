## 🎮&nbsp;&nbsp;구현 기능

- 10개의 카드가 무한히 회전
- 별도의 이동 아이콘클릭으로 카드 이동
- 정해진 시간이 되면 자동 이동
- 드래그를 통해 카드 이동
- 반응형 4단계로 구현
